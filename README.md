# Expense Tracker App
A simple and intuitive expense tracker built with Flutter. Visualize financial habits with a clean UI and seamless dark/light mode support.

## Features
✅ Dark/Light Mode
✅ Simple UI
✅ Visible Graph
✅ Track Expenses
✅ Expense Categories
